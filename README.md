# 疫情防控下医院人员调动系统 python978

## 项目概述

本项目是基于 Python 的疫情防控下医院人员调动系统，整合了前后端技术，主要包括首页、个人中心、医护人员管理等多个功能模块，以应对疫情下医院人员调动的需求。

## 技术栈

- **后端**: Python, Django
- **前端**: Vue
- **数据库**: MySQL
- **开发环境**: PyCharm
- **Python 版本**: 3.7

## 功能模块

- 首页
- 个人中心
- 医护人员管理
- 操作员管理
- 体温数据管理
- 隔离治疗管理
- 轮班调度管理
- 支援信息管理

## 系统角色

系统主要涉及以下角色：
- 医护人员
- 操作员
- 管理员

## 运行环境

- Python 3.7+
- MySQL
- Node.js（前端 Vue 项目运行环境）

## 部署步骤

1. 准备 Python 3.7+ 环境及 MySQL 数据库环境。
2. 克隆后端代码到本地。
3. 使用 PyCharm 或其他 IDE 打开后端项目，并配置数据库。
4. 前端 Vue 项目打包后部署至 Web 服务器。
5. 启动后端服务。

## 目录结构

```
项目目录/
├── backend/                # 后端 Django 项目目录
│   ├── apps/               # Django 应用目录
│   ├── config/             # 配置文件目录
│   ├── db/                 # 数据库相关文件目录
│   ├── media/              # 媒体文件目录
│   ├── static/             # 静态文件目录
│   ├── templates/          # 模板文件目录
│   └── ...
└── frontend/               # 前端 Vue 项目目录
    ├── node_modules/       # Node.js 依赖包目录
    ├── public/             # 公共资源目录
    ├── src/                # Vue 项目源码目录
    └── ...
```
